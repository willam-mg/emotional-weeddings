# Redesign Brief

Generated: 2026-06-30

## Creative Direction

Emotional Weddings should become a photography-first, editorial luxury wedding website for couples who value emotional storytelling, quiet observation, and refined documentary imagery.

The site should feel cinematic and human: composed, spacious, elegant, and emotionally present.

The strongest creative territory is:

> A quiet editorial eye for unrepeatable wedding moments.

## Brand Positioning

Emotional Weddings is not a generic wedding photography service. It should be positioned as a London-based editorial and documentary wedding photography studio for emotionally intelligent couples who want images that feel honest, stylish, and timeless.

Positioning pillars:

- Quiet observation
- Editorial composition
- Documentary emotion
- London and UK credibility
- Destination readiness
- Human warmth
- Premium but not cold

## Experience Principles

| Principle | Meaning |
| --- | --- |
| Photography leads | Every major page begins with visual evidence. |
| Copy breathes | Text is short, selective, and emotionally useful. |
| Trust is layered | Portfolio, Greg's story, process, FAQ, and contact all build confidence. |
| Services feel guided | Collections are framed by value and fit, not only price. |
| Inquiry is natural | CTAs should feel like an invitation, not pressure. |
| Mobile is editorial | Mobile layouts must feel designed, not merely stacked. |

## Messaging Priorities

Primary message:

Emotional, editorial wedding photography for couples who want their day remembered with honesty, elegance, and feeling.

Supporting messages:

- Greg works quietly and attentively.
- The photography blends documentary truth with editorial composition.
- The experience is personal, calm, inclusive, and considered.
- London-based, available across the UK and worldwide.
- Collections are crafted for different wedding scales and stories.

## Visual Priorities

1. Replace the blank/loading first viewport with a finished hero.
2. Use strong hero imagery and less explanatory text.
3. Fix or replace dark/empty gallery blocks.
4. Create consistent section spacing.
5. Use black/ivory/gold with restraint.
6. Give Services a calmer editorial flow before pricing.
7. Make Contact feel personal and clear.

## Technical Strategy

- Continue using WordPress and Divi.
- Use Divi Theme Builder for global header/footer.
- Create reusable Divi sections for hero, editorial content, CTA, testimonials, collections, and gallery preview.
- Keep `.env`, `wp-config.php`, SQL backups, uploads, cache, and logs out of Git.
- Use Playwright for screenshots and smoke tests.
- Use authenticated screenshots while maintenance mode is active.
- Clear/disable cache during design QA.

## First Redesign Implementation Target

Start with the Home page because it defines the system:

1. Fix first viewport blank/loading problem.
2. Create a completed editorial hero.
3. Keep the "quiet eye" positioning.
4. Replace incomplete gallery preview with reliable selected images.
5. Add a stronger inquiry CTA before the footer.

## Definition Of Success

The redesigned site succeeds when:

- A visitor understands the premium/emotional positioning in the first 10 seconds.
- The strongest images are visible before heavy text.
- The Services page feels premium, not transactional.
- Gallery and Contact become clear conversion bridges.
- Mobile feels intentionally designed.
- Playwright tests pass for public pages and admin login.
- Another agent can continue implementation from the documentation without needing basic clarification.
